# Sales and Demand Forecasting for Businesses

Machine Learning project developed for Future Interns ML Internship.

## Project Overview
This project predicts future sales and demand using historical sales data. It helps businesses make better inventory, marketing, and operational decisions.

## Features
- Sales Forecasting
- Demand Prediction
- Interactive Dashboard
- Business Insights
- Data Visualization
- Trend Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Plotly
- Matplotlib

## Dataset
Historical sales dataset used for forecasting future demand.

## Business Benefits
- Better inventory planning
- Improved demand forecasting
- Reduced stock shortages
- Data-driven business decisions
